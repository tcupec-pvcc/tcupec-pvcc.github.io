Networking & The Internet Terminology

TERM

DEFINITION
Machine Language A encoding system that consist of sequences of instructions that are encoded as numeric digits.
Debugging Process of locating and correcting errors found with a machine language.
Assembler Programs used to convert mnemonic expressions into machine language.
Assembly Language A mnemonic system for representing programs.
Translator A program that translated programs expressed high-level primitives into machine-language programs.
Compiler A program which translates several machine instructions into short sequences to simulate the activity requested by a single high-level primitive
Interpreter Programs that execute instructions as they were translated instead of recording the translated version for future use.
Cross-platform software Software that is independent of an operating system's design as well as the machine's hardware design and is therefore executable throughout a network.
Natural Language Language that evolved over time without formal grammatical analysis (ie English, German, and Latin)
Formal Language Language that is precisely defined by grammars (ie programming languages)
Machine independence The ability for a third-generation language to be compiled as easily for one machine as for another as the language used does not refer to the attributes of any particular machine.
High-level Programming Language Provide more abstractions for the programmer and make it easier for people to read and write a program.
Object-oriented programming (OOP) A programming paradigm in which a software system is viewed as a collection of units, called objects, each of which is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects.
Scripting language The collection of languages that are a subset of the imperative programming languages.
Data Types:
  1. Integer
  2. Float
  3. String
  1. Refers to numeric data consisting of whole numbers.
  2. Refers to numeric data that might contain values other than whole numbers.
  3. Refers to a sequence of characters consisting of symbols.
Primitive Data Type Data types that are included as primitives in a programming language, such as int for integer and char for character.
Data Structure The conceptual shape or arrangement of data.
Python Collection Data Structures
  1. List
  2. Tuple
  3. Set
  4. Dictionary
  1. Used to store multiple items in a single variable and are created using square brackets.
  2. A collection which is ordered and unchangeable and are written with round brackets.
  3. Unordered, no duplicates, unchangeable, and unidexed. Written with curly brackets.
  4. Used to store data values in key: value pairs. With curly brackets, keys and values. Ordered, changeable, and do not allow duplicates.
Function Is a block of code which only runs when it is called.
Local Variable A variable within a function; meaning that it can be referenced only within that function.
Global Variable A variable whose scope is not restricted to a particular part of a program
Source Program Program in its original form before it is translated from one language to another.
Object Program The translated version of program that has been translated from another language.
