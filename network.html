DEFINITION

Network Classifications

  • LAN
  • MAN
  • WAN
    		•
  • Local Area Network: A collection of devices connected together in one physical location, such as a building, office, or home.
  • Metropolitan Area Network: A computer network that connects computers within a metropolitan area, which could be a single large city, multiple cities and towns, or any given large area with multiple buildings.
  • Wide Area Network: Any large network that spreads out over a wide geographic area is a WAN. The Internet itself is considered a WAN.
    		•

    Network Models

  • Bus network
  • Star network
    		•
  • A local area network (LAN) topology in which each node -- a workstation or other device -- is connected to a main cable or link called a bus.
  • A star network is a local area network (LAN) topology in which all nodes -- personal computers (PCs), workstations or other devices -- are directly connected to a common central computer that is often referred to as a hub.
    		•

    Protocols

  • CSMA/CD Ethernet
  • CSMA/CA Wifi
    		•
  • Carrier Sence, Multiple Access with Collision Detection: A protocol that controls the right to transmit messages. CSMA/CD dictates that each message be broadcast to all the machines on the bus.
  • Carrier Sense, Multiple Access with Collision Avoidance: A policy, adopted by wireless networks, of trying to avoid transmission collisions rather than trying to detect them.

    • Combining Networks

  • Repeater
  • Bridge
  • Switch
  • Router
  • Gateway
    		•
  • An electronic device that receives a signal and retransmits it. Repeaters are used to extend transmissions so that the signal can cover longer distances or be received on the other side of an obstruction.
  • Used to connect two existing networks, it connects two buses by looking at the destination address that accompanies each message and fowards a message across the connection only when that message is destined for a computer on the other side.
  • Used to connect several existing networks, it produces a network consisting of several buses by considering the destination addresses of all messages and forwards only those messages destined for other buses.
  • The device that handles the connection between networks to form an internet; essentially special purpose computers used for forwarding messages.
  • A network node used in telecommunications that connects two networks with different transmission protocols together.

    Internet Four-Layer Software

  • Application layer
  • Transport layer
  • Network layer
  • Link layer
    		•
  • Consists of those software units such as clients and servers that use Internet Communication to carry out their tasks. This where a message typically originates.
  • Its task is to accept messages from the application layer and to ensure that the messages are properly formatted for transmission over the Internet.
  • Its task is to decide in which direction a packet, received from the transport layer, should be sent at each step along the packet's path through the Internet.
  • Its task is to actually transmit the packet up to its netowrk layer where the packet's final desitination is compared to the network layer's forwarding table to determine the direction of the packet's next step.
    		•

    Cybersecurity

  • Malware
  • Virus
  • Worm
  • Trojan horse
  • Spyware
  • Phishing
  • DDos attack
    		•
  • The collective name of malicious software used to attack networks.
  • Software that infects a computer by inserting itself into programs that already reside in the machine.
  • An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
  • A program that enteres a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim.
  • Another form of malicious software (sometimes called sniffing software), that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
  • A technique of obtaining information explicitly by simply asking for it.
  • Denial of Service - The process of overloading a computer with messages.
